This page must run through the dev server

Opening index.html directly uses a file:// URL. This app is built with Vite + React; the browser cannot load /src/main.jsx from disk that way, so the area below stays blank.

Do this instead:

  1. Open a terminal in this project folder (where package.json lives).
  2. Run npm install once if you have not already.
  3. Run npm run dev.
  4. Open the URL Vite prints (usually http://localhost:5173).

Static build: npm run build then npm run preview.
